# ComputorV2

Implementation of a command line mathematical parser, using python 3.10

```
> python computorv2.py
```

## Assignation

Save variables or functions. Supports Real and Imaginary numbers, Matrices.

```
> a = 3i + 2
2.0+3.0i
> f(x) = 3x^2 - cos(x)
```

## Calculation

Evaluate expressions.

```
>2x(3x+2) + sin(43) = ?
>f(x) = ?
>f(3) = ?
```
